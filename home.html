What is Structure Function?

Structure Function

Five tags used to describe the structure of academic text and the function of sections.

  • introduction
  • related work
  • method
  • result
  • conclusion

We could tag sections' function with these five tags, and a unified structure could be more meaningful.

Some applications: citation analysis with structure function; academic search with paper structure; even Mobile Reading!

How?

Recognition Methods

We solve the recognition in three levels:

  • Section Header Based demo »

    • Sequence tagging

    The section headers in a literature are viewed as a sequence with hidden structure function tags.

  • Section Content Based demo »

    • Text Categorization

    Using the section content to improve the accuracy of header based method.

  • Paragraph Based demo »

    • Two subtasks:

    • Paragraph position recognition

      • Text Categorization

      Given a paragraph of a literature, to recognize which structure function it belong to.

    • Section function recognition

      • Majority Voting

      Recognize the function of section based on results of paragraph position recognition with majority voting.
